# Motion SVG

Веб‑редактор для создания автономных анимированных SVG‑иконок. Импортирует SVG, распознаёт верхнеуровневые графические элементы, позволяет назначить анимацию всей иконке или отдельным слоям и скачать готовый файл.

## Возможности

- загрузка кликом или перетаскиванием `.svg`;
- выбор одного или нескольких слоёв;
- поворот, сдвиг, исчезание, масштаб, прыжок и прорисовка контура;
- длительность, задержка, easing, амплитуда, направление и число повторов;
- пауза и перезапуск предпросмотра;
- экспорт автономного SVG с встроенными CSS keyframes;
- удаление потенциально опасных `script`, `foreignObject` и inline-обработчиков при импорте.

## Запуск

Требуется Node.js `>=22.13.0`.

```bash
npm install
npm run dev
```

Откройте `http://localhost:3000`.

## Проверка

```bash
npm run build
npm test
```

Основной интерфейс находится в `app/page.tsx`, стили — в `app/globals.css`.
